# Russald Medical Center — Web

**Sitio:** https://russaldmedical.com  
**Stack:** Next.js 15 · App Router · Tailwind CSS v4 · TypeScript strict · Framer Motion  
**Deploy:** Vercel (auto-deploy en `main`)

## Guía operativa

Ver [CLAUDE.md](./CLAUDE.md) — fuente de verdad para arquitectura, convenciones de código, tokens, Figma node IDs y backlog.

## Inicio rápido

```bash
pnpm install
pnpm dev
```

## Verificación antes de PR

```bash
pnpm exec tsc --noEmit   # type check
pnpm build               # build completo
```
